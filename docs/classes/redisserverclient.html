    • Defined in lib/client.ts:81
    • Redis官方文档：https://redis.io/commands/quit
    • 起始版本：1.0.0

    请求 Redis server 关闭连接。

    始终返回 OK。

    Returns Promise<void>

select

  • select(db: number): Promise<string>
    • Defined in lib/client.ts:95
    • 起始版本：1.0.0

    选择从 0 开始计数的 Redis 逻辑数据库。

    Redis 的可选数据库是一种命名空间格式。所有的数据仍然存在相同的 RDB / AOF 文件中。不同的数据库可以有相同的 key。
    像 FLUSHDB [[RedisClient.swapdb | SWAPDB]] RANDOMKEY 可以在指定的数据库工作。

    Parameters

    • db: number

    Returns Promise<string>
