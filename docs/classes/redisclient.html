expire

  • expire(key: string, ttl: number): Promise<0 | 1>
    • Defined in redis-client.ts:89
    起始版本：1.0.0
时间复杂度：O(1)

    设置 key 的超时时间，到期后 key 会自动删除。

    当 删除 key 或 重写 值的时候，包括 DEL，SET，GETSET 和所有 *STORE 命令，超时时间设置会被清除。 其他的诸如，INCR，LPUSH 等 概念上修改 了值的操作，不会影响超时时间。

    Parameters

    • key: string

      需要查询的 key

    • ttl: number

      需要设置的超时时间

    Returns Promise<0 | 1>

    执行结果：1 ttl 设置成功，0 key 不存在，设置失败。

    查看原始定义

pttl

  • pttl(key: string): Promise<number>
    • Defined in redis-client.ts:67
    起始版本：2.6.0
时间复杂度：O(1)

    返回查询的值的剩余有效毫秒数。

    • 如果 key 不存在，2.8 及之后的版本返回 -2， 2.6 及以前，返回 -1。
    • 如果 key 存在且未设置 ttl 则返回 -1。

    Parameters

    • key: string

      需要查询的 key

    Returns Promise<number>

    秒为单位的 TTL，返回值为负数时，表示错误信号。

    查看原始定义
